## Overview

The `Hoverable` utility provides a 3D tilt effect that responds to mouse movement, creating an engaging interactive experience. It applies perspective transforms to elements based on cursor position.

## Usage

Import the `Hoverable` utility and apply its event handlers to any element:

```tsx
import { component$ } from '@builder.io/qwik';
import { Hoverable } from '@luminescent/ui-qwik';

export default component$(() => {
  return (
    <div
      class="w-64 h-64 bg-gradient-to-br from-blue-500 to-purple-600 rounded-lg"
      onMouseMove$={Hoverable.onMouseMove$}
      onMouseLeave$={Hoverable.onMouseLeave$}
    >
      Hover over me!
    </div>
  );
});
```

## How It Works

The `Hoverable` utility calculates the mouse position relative to the element and applies a 3D rotation transform:

```typescript
export const Hoverable = {
  onMouseMove$: (e: MouseEvent, el: HTMLElement) => {
    const rect = el.getBoundingClientRect();
    const mouseX = e.clientX - rect.left;
    const mouseY = e.clientY - rect.top;

    const rotateX = ((mouseY / rect.height) - 0.5) * -10;
    const rotateY = ((mouseX / rect.width) - 0.5) * 10;

    el.style.transition = 'transform 0.05s ease-out';
    el.style.transform = `perspective(500px) rotateX(${rotateX}deg) rotateY(${rotateY}deg)`;
  },
  onMouseLeave$: (_e: MouseEvent, el: HTMLElement) => {
    el.style.transition = 'transform 0.3s ease-out';
    el.style.transform = 'perspective(500px) rotateX(0deg) rotateY(0deg)';
  },
} as const;
```

## Behavior

### onMouseMove$

- Calculates mouse position relative to element bounds
- Converts position to rotation angles (max ±5° on each axis)
- Applies smooth transform with 50ms transition
- Uses 500px perspective for 3D effect

### onMouseLeave$

- Resets rotation to neutral position (0° on both axes)
- Uses slower 300ms transition for smooth return animation
- Maintains same perspective value

## Advanced Examples

### Card with Hoverable Effect

```tsx
import { component$ } from '@builder.io/qwik';
import { Hoverable } from '@luminescent/ui-qwik';

export const HoverCard = component$(() => {
  return (
    <div
      class="relative w-80 h-96 rounded-2xl bg-gradient-to-br from-slate-900 to-slate-800 p-8 shadow-xl"
      onMouseMove$={Hoverable.onMouseMove$}
      onMouseLeave$={Hoverable.onMouseLeave$}
    >
      <h2 class="text-2xl font-bold text-white mb-4">Interactive Card</h2>
      <p class="text-gray-300">
        Move your mouse over this card to see the 3D tilt effect in action.
      </p>
    </div>
  );
});
```

### Grid of Hoverable Items

```tsx
import { component$ } from '@builder.io/qwik';
import { Hoverable } from '@luminescent/ui-qwik';

export default component$(() => {
  const items = [
    { title: 'Item 1', color: 'from-blue-500 to-cyan-500' },
    { title: 'Item 2', color: 'from-purple-500 to-pink-500' },
    { title: 'Item 3', color: 'from-green-500 to-emerald-500' },
    { title: 'Item 4', color: 'from-orange-500 to-red-500' },
  ];

  return (
    <div class="grid grid-cols-2 gap-6 p-8">
      {items.map((item) => (
        <div
          key={item.title}
          class={`h-48 rounded-xl bg-gradient-to-br ${item.color} p-6 flex items-center justify-center text-white text-xl font-bold`}
          onMouseMove$={Hoverable.onMouseMove$}
          onMouseLeave$={Hoverable.onMouseLeave$}
        >
          {item.title}
        </div>
      ))}
    </div>
  );
});
```

## Customization

You can adjust the intensity of the effect by modifying the rotation multipliers:

```typescript
// Default uses ±10 degrees (multiplied by 0.5 offset = ±5 degrees max)
const rotateX = ((mouseY / rect.height) - 0.5) * -10;
const rotateY = ((mouseX / rect.width) - 0.5) * 10;

// For stronger effect, increase the multiplier:
const rotateX = ((mouseY / rect.height) - 0.5) * -20;  // ±10 degrees
const rotateY = ((mouseX / rect.width) - 0.5) * 20;

// For subtler effect, decrease the multiplier:
const rotateX = ((mouseY / rect.height) - 0.5) * -5;   // ±2.5 degrees
const rotateY = ((mouseX / rect.width) - 0.5) * 5;
```

## Performance Notes

- Uses hardware-accelerated CSS transforms for smooth performance
- Minimal JavaScript execution on each mouse move
- No external dependencies beyond Qwik
- Works on all modern browsers with CSS transform support

## Accessibility

The `Hoverable` effect is purely visual and doesn't affect functionality:

- Works with keyboard navigation (no hover effect, but element remains functional)
- No impact on screen readers
- Respects `prefers-reduced-motion` when used with appropriate CSS

```css
@media (prefers-reduced-motion: reduce) {
  * {
    transition: none !important;
    transform: none !important;
  }
}
```
