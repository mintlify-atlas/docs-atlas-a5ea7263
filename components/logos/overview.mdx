## Overview

Luminescent UI includes a collection of optimized SVG logo components for popular Minecraft-related projects and platforms. All logos support customizable sizing and inherit color from the current text color using `fill="currentColor"`.

## Basic Usage

Import and use logo components like any other Qwik component:

```tsx
import { LogoPaper, LogoDiscord, LogoLuminescent } from '@luminescent/ui-qwik';

export default component$(() => {
  return (
    <div class="flex items-center gap-4">
      <LogoPaper size={32} />
      <LogoDiscord size={32} />
      <LogoLuminescent size={32} />
    </div>
  );
});
```

## IconProps Interface

All logo components extend the `IconProps` interface:

```typescript
import type { PropsOf } from '@builder.io/qwik';

export type IconProps = Omit<PropsOf<'svg'>, 'height' | 'width'> & {
  size?: number;
}
```

This interface:
- Accepts all standard SVG attributes except `height` and `width`
- Provides a `size` prop that sets both width and height
- Supports class names, styles, event handlers, and other SVG props

## Styling with Color

Logos use `fill="currentColor"` or `stroke="currentColor"`, so they inherit the text color:

```tsx
<div class="text-blue-500">
  <LogoPaper size={48} />
</div>

<div class="text-purple-600">
  <LogoDiscord size={48} />
</div>
```

## Special Logo Features

### LogoBirdflop

The Birdflop logo has additional props for customization:

```tsx
import { LogoBirdflop } from '@luminescent/ui-qwik';

// Basic usage
<LogoBirdflop size={64} />

// With confused state
<LogoBirdflop size={64} confused />

// With gradient fill
<LogoBirdflop 
  size={64} 
  fillGradient={['#1db4f4', '#ad83f4']} 
/>
```

**Props:**
- `confused?: boolean` - Displays a question mark overlay
- `fillGradient?: string[]` - Array of color stops for a linear gradient

### LogoLuminescent

The Luminescent logo comes in two variants:

```tsx
import { 
  LogoLuminescent,      // Icon only
  LogoLuminescentFull   // Full wordmark
} from '@luminescent/ui-qwik';

<LogoLuminescent size={48} />
<LogoLuminescentFull size={48} />
```

Note: `LogoLuminescentFull` uses `height` instead of `width` for sizing to maintain aspect ratio.
