## Overview

The SelectMenu component provides a styled alternative to native select elements with support for custom dropdowns, hover interactions, and flexible positioning. It includes both a labeled wrapper (`SelectMenu`) and a raw version (`SelectMenuRaw`) without label.

## Import

```tsx
import { SelectMenu, SelectMenuRaw } from '@luminescent/ui-react';
```

## Props

### SelectMenu Props

The labeled wrapper component that includes all SelectMenuRaw props plus:

<ParamField path="children" type="React.ReactNode" required>
  The label text to display above the select menu.
</ParamField>

<ParamField path="id" type="string" required>
  HTML id attribute for the select element (used for label association).
</ParamField>

### SelectMenuRaw Props

<ParamField path="values" type="Array<{ name: React.ReactNode; value: string | number }>">
  Array of options to display in the dropdown menu.
  
  ```tsx
  values={[
    { name: 'Label 1', value: 'value1' },
    { name: <span>Custom JSX</span>, value: 'value2' }
  ]}
  ```
</ParamField>

<ParamField path="customDropdown" type="boolean" default="false">
  When true, uses the custom `dropdown` prop content instead of the selected value's name.
</ParamField>

<ParamField path="dropdown" type="React.ReactNode">
  Custom content to display in the dropdown button. Used when `customDropdown` is true or when `values` is not provided.
</ParamField>

<ParamField path="extra-buttons" type="React.ReactNode">
  Additional buttons or content to render at the bottom of the dropdown panel.
</ParamField>

<ParamField path="hover" type="boolean" default="false">
  When true, opens the dropdown on hover instead of click.
</ParamField>

<ParamField path="align" type="'left' | 'right' | 'center'">
  Horizontal alignment of the dropdown panel relative to the button.
</ParamField>

<ParamField path="nocloseonclick" type="boolean" default="false">
  When true, prevents automatic closing when clicking the button.
</ParamField>

<ParamField path="noblur" type="boolean" default="false">
  When true, disables the backdrop blur effect on the dropdown panel.
</ParamField>

<ParamField path="panelClass" type="string" default="'lum-bg-lum-input-bg'">
  CSS class for the dropdown panel background.
</ParamField>

<ParamField path="btnClass" type="string" default="'lum-bg-transparent'">
  CSS class for individual option buttons.
</ParamField>

<ParamField path="className" type="string">
  Additional CSS classes for the dropdown button.
</ParamField>

<ParamField path="value" type="string | number">
  Controlled value of the select element.
</ParamField>

## TypeScript Interface

```tsx
interface SelectMenuProps
  extends React.SelectHTMLAttributes<HTMLSelectElement> {
  customDropdown?: boolean;
  panelClass?: string;
  btnClass?: string;
  noblur?: boolean;
  nocloseonclick?: boolean;
  hover?: boolean;
  align?: 'left' | 'right' | 'center';
  values?: {
    name: React.ReactNode;
    value: string | number;
  }[];
  dropdown?: React.ReactNode;
  'extra-buttons'?: React.ReactNode;
}
```

## Behavior

### Selection Handling

When an option is clicked:
1. The dropdown closes (unless `nocloseonclick` is true)
2. The hidden select element's value is updated
3. A `change` event is dispatched on the select element
4. The displayed value updates to show the selected option

### Click Outside Detection

The dropdown automatically closes when:
- Clicking outside the component
- Clicking an option (unless `nocloseonclick` is true)
- The close listener checks the event's composed path to determine if the click is outside

### Hover Mode

When `hover` is true:
- Click events are ignored
- Dropdown opens on hover
- A 2-unit height spacer is added for better UX
- Faster transition timing (75ms)

## Alignment Options

```tsx
{/* Left aligned (default) */}
<SelectMenuRaw align="left" values={options} />

{/* Right aligned */}
<SelectMenuRaw align="right" values={options} />

{/* Center aligned */}
<SelectMenuRaw align="center" values={options} />
```

- **Left**: `left-0`
- **Right**: `right-0`
- **Center**: `left-1/2 -translate-x-1/2`

## Styling

### Custom Panel Styles

```tsx
<SelectMenu
  id="custom"
  panelClass="bg-slate-900"
  btnClass="hover:bg-slate-800"
  values={options}
>
  Custom styled menu
</SelectMenu>
```

### Dropdown Features

- Maximum height: `max-h-72` (288px)
- Scrollable: `lum-scroll overflow-auto`
- Rounded corners: `rounded-lum`
- Border with gap spacing: `gap-1 border p-1`
- Animation: scale and opacity transitions

## Accessibility

- Uses a hidden native `<select>` element that syncs with the custom UI
- Supports standard select attributes (name, required, etc.)
- Dispatches native change events
- Keyboard accessible through browser's focus management
- Non-selectable dropdown text: `select-none`

## Examples

### With Change Handler

```tsx
import { SelectMenu } from '@luminescent/ui-react';
import { useState } from 'react';

export function ControlledSelect() {
  const [value, setValue] = useState('option1');

  return (
    <div>
      <SelectMenu
        id="controlled"
        value={value}
        onChange={(e) => setValue(e.target.value)}
        values={[
          { name: 'Option 1', value: 'option1' },
          { name: 'Option 2', value: 'option2' },
          { name: 'Option 3', value: 'option3' }
        ]}
      >
        Select an option
      </SelectMenu>
      <p>Selected: {value}</p>
    </div>
  );
}
```

## Component Relationship

- `SelectMenu` wraps `SelectMenuRaw` and adds a label
- `SelectMenuRaw` uses the `Dropdown` component internally
- Both components share the same core props and functionality
