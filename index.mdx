## Quick start

<Steps>
  <Step title="Install the core package">
    Install the Luminescent UI library from npm using your preferred package manager.

    <CodeGroup>
    ```bash npm
    npm install @luminescent/ui
    ```
    ```bash pnpm
    pnpm install @luminescent/ui
    ```
    ```bash yarn
    yarn add @luminescent/ui
    ```
    </CodeGroup>
  </Step>

  <Step title="Add to your Tailwind CSS">
    Import the Luminescent UI styles and plugin into your main CSS file.

    ```css
    @import "@luminescent/ui";
    @plugin "@luminescent/ui/lum-bg";
    ```
  </Step>

  <Step title="Start using components">
    Install framework-specific packages for React or Qwik components.

    <Tabs>
      <Tab title="Qwik">
        ```bash
        pnpm install @luminescent/ui-qwik
        ```

        Add to your CSS file:
        ```css
        @config "@luminescent/ui-qwik/config";
        @source "../node_modules/@luminescent/ui-qwik";
        ```
      </Tab>
      <Tab title="React">
        ```bash
        pnpm install @luminescent/ui-react
        ```

        Add to your CSS file:
        ```css
        @source "../node_modules/@luminescent/ui-react";
        ```
      </Tab>
    </Tabs>
  </Step>
</Steps>
