Luminescent UI provides a powerful set of custom CSS utilities built on top of Tailwind CSS. These utilities enable advanced styling with dynamic borders, depth effects, and color mixing.

## Background Utilities

### lum-bg-*

The `lum-bg-*` utility system provides intelligent background colors with automatic border generation and depth effects.

<CodeGroup>
```css index.css
@utility lum-bg-* {
  /* Base color logic */
  --alpha: --modifier(integer);
  --bg-color: --alpha(--value([color]) / calc(var(--alpha, 100) * 1%));
  --bg-color: --alpha(--value(--color-*) / calc(var(--alpha, 100) * 1%));
  --border-color: color-mix(in oklab, var(--color-lum-border) var(--lum-border-mix), var(--bg-color));

  @apply border border-transparent outline-none;

  &:focus {
    border-color: var(--color-lum-accent);
  }

  /* Border fades out when depth is active */
  border-color: color-mix(
    in oklab,
    var(--border-color),
    transparent calc(var(--lum-depth) * 100%)
  );

  /* Shadow fades in when depth is active */
  box-shadow:
    inset 0 calc(1px * var(--lum-depth)) calc(2px * var(--lum-depth)) var(--border-color),
    0 calc(1px * var(--lum-depth)) calc(2px * var(--lum-depth)) rgba(0, 0, 0, 0.05),
    0 calc(1px * var(--lum-depth)) calc(3px * var(--lum-depth)) rgba(0, 0, 0, 0.1);

  background-color: var(--bg-color);
}
```

```html Usage
<div class="lum-bg-blue-500">Blue background</div>
<div class="lum-bg-red-400/50">Red background with 50% opacity</div>
<div class="lum-bg-luminescent-600">Brand color background</div>
```
</CodeGroup>

<Note>
  The `lum-bg-*` utility automatically generates contextual borders using `color-mix()` and supports alpha modifiers (e.g., `lum-bg-blue-500/50`).
</Note>

#### Features

- **Automatic border generation**: Borders are automatically calculated by mixing `--color-lum-border` with the background color
- **Alpha modifier support**: Add `/50` or any percentage to control opacity
- **Depth effects**: Responds to `--lum-depth` variable for 3D shadow effects
- **Focus states**: Automatic focus border using `--color-lum-accent`
- **Color keywords**: Supports `inherit`, `transparent`, and `currentColor`

### lum-grad-bg-*

Creates gradient backgrounds with the same border and depth features as `lum-bg-*`.

<CodeGroup>
```css index.css
@utility lum-grad-bg-* {
  --bg-color: --alpha(--value([color]) / calc(var(--alpha, 100) * 1%));
  --gradient-color: color-mix(in oklab, var(--color-lum-gradient) var(--lum-gradient-mix), var(--bg-color));

  background: 
    linear-gradient(
      var(--lum-gradient-angle),
      var(--bg-color),
      var(--gradient-color)
    );
}
```

```html Usage
<div class="lum-grad-bg-purple-500">
  Gradient from purple to dark
</div>
```
</CodeGroup>

<Accordion title="Gradient Customization Variables">
  - `--lum-gradient-mix`: Controls the intensity of the gradient (default: `20%`)
  - `--lum-gradient-angle`: Direction of the gradient (default: `180deg`)
  - `--color-lum-gradient`: Base color to mix with (default: `var(--color-gray-950)`)
</Accordion>

### lum-bg (Base)

A base utility that works with a pre-defined `--bg-color` variable.

```html
<div class="lum-bg" style="--bg-color: oklch(0.7 0.2 270)">
  Custom OKLCH color
</div>
```

### lum-toggle-bg-*

Specialized background utility for toggle/button states with hover effects.

```css
@utility lum-toggle-bg-* {
  /* Same base as lum-bg-* */
  
  &::after, &:hover {
    background-color: --alpha(color-mix(in oklab, white 10%, --value(--color-*)) / calc(var(--alpha, 100) * 1%));
  }
}
```

```html
<button class="lum-toggle-bg-blue-500">
  Hover me for lighter shade
</button>
```

## Border Utilities

### rounded-lum

Applies Luminescent's border radius with superellipse corner shaping.

```css
@utility rounded-lum {
  border-radius: var(--lum-border-radius);
  corner-shape: superellipse(var(--lum-border-superellipse));
}
```

```html
<div class="rounded-lum lum-bg-gray-800">
  Smooth superellipse corners
</div>
```

### rounded-lum-*

Subtracts spacing from the border radius for nested elements.

```html
<div class="rounded-lum p-4">
  <div class="rounded-lum-1">
    Nested with reduced radius
  </div>
</div>
```

<Note>
  The `rounded-lum-*` utility subtracts `spacing * value` from the base border radius, perfect for nested components.
</Note>

### border-gradient-*

Creates gradient borders using the `::before` pseudo-element.

<CodeGroup>
```css index.css
@utility border-gradient-* {
  @apply relative box-border bg-clip-padding;
  border-width: calc(--value(integer) * 1px);
  border-color: transparent !important;
  &::before {
    @apply bg-gradient-to-br content-[''] absolute inset-0 z-[-1];
    margin: calc(--value(integer) * -1px);
    border-radius: inherit;
  }
}
```

```html Usage
<div class="border-gradient-2 from-blue-500 to-purple-500 rounded-lum p-4">
  Gradient border with 2px width
</div>
```
</CodeGroup>

## Spacing Utilities

### lum-btn-p-*

Intelligent button padding that scales horizontally based on `--lum-btn-p-x`.

```css
@utility lum-btn-p-* {
  padding: calc(var(--spacing) * --value(integer))
           calc(var(--spacing) * --value(integer) * var(--lum-btn-p-x))
           calc(var(--spacing) * --value(integer))
           calc(var(--spacing) * --value(integer) * var(--lum-btn-p-x));

  gap: calc(
    var(--spacing) * --value(integer) * clamp(0.65, 4 / --value(integer), 1)
  );
}
```

```html
<button class="lum-btn-p-2">Button</button>
<button class="lum-btn-p-3">Larger Button</button>
```

<Note>
  The gap is intelligently calculated with a clamp function to maintain proportions at different sizes.
</Note>

### lum-input-p-*

Similar to button padding but uses `--lum-input-p-x` for horizontal scaling.

```css
@utility lum-input-p-* {
  padding: calc(var(--spacing) * --value(integer))
    calc(var(--spacing) * --value(integer) * var(--lum-input-p-x))
    calc(var(--spacing) * --value(integer))
    calc(var(--spacing) * --value(integer) * var(--lum-input-p-x));
}
```

```html
<input class="lum-input-p-2" />
<textarea class="lum-input-p-3"></textarea>
```

## Component Utilities

### lum-btn

Complete button styling with transitions and interactions.

```css
@utility lum-btn {
  @apply flex items-center motion-safe:transition duration-300 
         hover:duration-75 active:duration-75 ease-out disabled:opacity-50 
         hover:drop-shadow-lg active:drop-shadow-lg motion-safe:active:scale-95 
         whitespace-nowrap touch-manipulation select-none lum-btn-p-2 
         text-base lum-bg-lum-input-bg hover:lum-bg-lum-input-hover-bg 
         active:lum-bg-lum-input-hover-bg;
  border-radius: var(--lum-border-radius);
  corner-shape: superellipse(var(--lum-border-superellipse));
}
```

```html
<button class="lum-btn">Click me</button>
<a href="#" class="lum-btn">Button link</a>
```

### lum-input

Standardized input styling.

```css
@utility lum-input {
  @apply flex motion-safe:transition duration-300 hover:duration-75 
         active:duration-75 ease-out disabled:opacity-50 
         hover:drop-shadow-lg active:drop-shadow-lg whitespace-nowrap 
         touch-manipulation select-none lum-input-p-2 lum-bg-lum-input-bg 
         hover:lum-bg-lum-input-hover-bg active:lum-bg-lum-input-hover-bg;
  border-radius: var(--lum-border-radius);
  corner-shape: superellipse(var(--lum-border-superellipse));
}
```

```html
<input type="text" class="lum-input" placeholder="Enter text..." />
<select class="lum-input">...</select>
```

### lum-card

Card container with consistent spacing and background.

```css
@utility lum-card {
  @apply flex flex-col gap-3 whitespace-pre-wrap lum-bg-lum-card-bg p-7;
  border-radius: var(--lum-border-radius);
  corner-shape: superellipse(var(--lum-border-superellipse));
}
```

```html
<div class="lum-card">
  <h3>Card Title</h3>
  <p>Card content with consistent styling</p>
</div>
```

### lum-hoverable

Adds hover scale and shadow effects.

```css
@utility lum-hoverable {
  @apply transition hover:scale-102 active:scale-102 
         hover:drop-shadow-lg active:drop-shadow-lg 
         hover:duration-75 active:duration-75 ease-out;
}
```

```html
<div class="lum-card lum-hoverable">
  Hover me for interaction feedback
</div>
```

### lum-loading

Loading spinner utility.

```css
@utility lum-loading {
  @apply animate-spin border-transparent border-l-current 
         border-t-current border-3 rounded-full;
}
```

```html
<div class="lum-loading w-8 h-8"></div>
```
